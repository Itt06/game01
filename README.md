# 神経衰弱

神経衰弱ゲームです。練習だからいろいろ試していたら、難易度が異常に高くなってしまい面白くなくなりました。

## 概要
52枚のカード（黒1〜13、赤1〜13）を使用し、色と数字が完全に一致するペアを当てる記憶力ゲームです。

## 特徴
- **フィッシャー・イェーツシャッフル**: 毎回完全にランダムな盤面を生成します。
- **ヒント機能**: 毎ターン、ランダムに2枚のカードの内容が表示されます。

## 遊び方
1. プログラムを実行すると、52枚のカードが「■」で表示されます。
2. 1枚目と2枚目の座標を `X Y` の形式で入力します（例: `1 1`）。
3. 色と数字が一致すればカードを獲得し、スコアが加算されます。
4. 設定された回数（デフォルト20回）以内にどれだけ多くのペアを見つけられるか挑戦してください。

## 開発環境
- 言語: C言語
- コンパイラ: gcc
